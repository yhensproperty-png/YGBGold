# YGB Buy Gold & Sell

## Manual Order Fulfillment Flow

Currently, the platform captures purchase intents via the "Request to Buy" feature but does not process payments directly. Instead, admins follow a manual fulfillment flow:

1. **Review Pending Requests**: When a customer clicks "Request to Buy," a new order appears in the "**Orders**" tab of the Admin Manage page with a `pending` status.
2. **Contact Customer**: Admins confirm orders manually via WhatsApp or Email using the contact details provided in the order request to arrange payment.
3. **Save Notes**: Admins can use the private **Admin Notes** field to save WhatsApp chat summaries, payment proofs, or any special arrangements related to the order.
4. **Confirm**: Once payment is verified, the admin clicks **Confirm** on the order entry.
5. **Ship & Track**: After dispatching the item to the buyer, the admin enters the courier's **Tracking #** and clicks **Mark Shipped**.
6. **Completion**: The customer can track their order through the provided courier, and the order remains securely recorded in the platform.
